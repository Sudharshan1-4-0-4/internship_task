
# TASK

This project is a web application built with React.js and Vite, aiming to design user friendly web application. It allows users to submit the assessment.


## Features

- Submit the assessment
- Cross platform


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sudharshan1-4-0-4/internship_task
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
->open another terminal

```bash
  cd backend
```
Start the server
```bash
  node index.js
```


## Tech Stack

**Client:** React, css

**Server:** Node, Express
